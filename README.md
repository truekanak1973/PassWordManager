## ⚙️ Setup Instructions

### 1. Clone the repository

```
git clone https://github.com/truekanak1973/PassWordManager.git
cd PassWordManager
```

---

### 2. Backend Setup

```
cd backend
npm install
```

Create a `.env` file inside `backend/` and add:

```
MONGO_URI=your_mongodb_connection_string
DB_NAME=passwordManager
```

---

### 3. Run the server

```
node server.js
```

Server runs at:

```
http://localhost:3000
```
